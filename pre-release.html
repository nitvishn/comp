Task One

Task Description

For every item in the auction, the item number, description and the reserve price should be recorded. The number of bids is set to zero. There must be at least 10 items in the auction.

Code

MinItems = 10

REPEAT
	OUTPUT "Enter the number of items:" 
	INPUT N
	IF (N < MinItems) THEN
    OUTPUT "There must be at least", MinItems, "items."
	END IF
UNTIL (N >= MinItems)

ItemNumber[1:N]
Description[1:N]
ReservePrice[1:N]
NumberBids[1:N]

FOR I = 1 TO N
  OUTPUT "Enter the item number for item", I, ": "
  INPUT ItemNumber[I]
	OUTPUT "Enter the description for item", I, ": "
	INPUT Description[I]
	OUTPUT "Enter the reserve price for item", I, ": "
	INPUT ReservePrice[I]
	NumberBids[I] = 0
NEXT I

Data Structures

  • MinItems
    • Data structure: Constant
    • Data type: Integer
    • Use: Minimum number of items that have to be present for auction
  • N
    • Data structure: Variable
    • Data type: Integer
    • Use: Number of items that are available for the auction
  • ItemNumber
    • Data structure: Array
    • Data type: Integer
    • Use: To store the item numbers of the items in the auction
  • Description
    • Data structure: Array
    • Data type: String
    • Use: To store the description of the items in the auction
  • ReservePrice
    • Data structure: Array
    • Data type: Currency
    • Use: To store the reserve price of the items in the auction
  • NumberBids
    • Data structure: Array
    • Data type: Integer
    • Use: To store the number of bids for each product in the auction

Task Two

Task Description

A buyer should be able to find an item and view the item number, description and the current highest bid. A buyer can then enter their buyer number and bid, which must be higher than any previously recorded bids. Every time a new bid is recorded the number of bids for that item is increased by one. Buyers can bid for an item many times and they can bid for many items.

Code

HighestBid[1:N]

FOR I = 1 TO N
	HighestBid[I] = 0
NEXT I

Choice = "Yes" 

WHILE (Choice = "Yes")
	OUTPUT "Enter the item number you would like to view: "
	INPUT BidItemNum
	FOR I = 1 TO N
		IF (ItemNumber[I] = BidItemNum) THEN
			OUTPUT "Item Number:", ItemNumber[I]
			OUTPUT "Item Description:", Description[I]
			OUTPUT "Highest Bid:", HighestBid[I]

			OUTPUT "Do you want to bid for this item? (Yes/No): "
			INPUT Choice

			IF (Choice = "Yes") THEN
				OUTPUT "Please enter your buyer number: "
				INPUT BuyerNumber

				REPEAT
					OUTPUT "Enter the new bid: "
					INPUT BidAmount
					IF (BidAmount <= HighestBid[I]) THEN
						OUTPUT "Please bid higher than the current highest bid."
					END IF
				UNTIL (BidAmount > HighestBid[I])

				HighestBid[I] = BidAmount
				NumberBids[I] = NumberBids[I] + 1
			END IF

			I = N + 2
		END IF
	NEXT I

	IF (NOT I = N + 3) THEN
		OUTPUT "Sorry, the item number you entered doesn't exist."
	ENDIF

	OUTPUT "Do you want to view/bid again for any item? (Yes/No): "
	INPUT Choice
END

Task Three

Task Description

Using the results from task two, identify items that have reached their reserve price, mark them as sold, calculate 10% of the final bid as the auction company fee and add this to the total fee for all sold items. Display this total fee. Display the item number and final bid for all the items with bids that have not reached their reserve price. Display the item number of any items that have received no bids. Display the number of items sold, the number of items that did not meet the reserve price and the number of items with no bids.

Code

Sold[1:N]
TotalFee[1:N]
CountSold = 0
AuctionRevenue = 0
BidsButUnsold = 0
CountNoBids = 0

OUTPUT "These are the items which were sold:"
FOR I = 1 TO N
	IF (HighestBid[I] >= ReservePrice[I]) THEN
		Sold[I] = True
		CountSold = CountSold + 1
		TotalFee[I] = 1.1*HighestBid[I]
		AuctionRevenue = AuctionRevenue + 0.1*HighestBid[I]
		OUTPUT "Item Number:", ItemNumber[I]
		OUTPUT "Description:", Description[I]
		OUTPUT "Total Fee:", TotalFee[I]
	ELSE
		Sold[I] = False
	END IF
NEXT I

OUTPUT "These are the items which were not bid for: "
FOR I = 1 TO N
	IF (NumberBids[I] = 0) THEN
		CountNoBids = CountNoBids + 1
		OUTPUT "Item Number:", ItemNumber[I]
	END IF
NEXT I

OUTPUT "These are the items with bids which didn't reach their reserve price: "
FOR I = 1 TO N
	IF (NumberBids[I] > 0 AND (NOT Sold[I])) THEN
		BidsButUnsold = BidsButUnsold + 1
		OUTPUT "Item Number:", ItemNumber[I]
		OUTPUT "Highest Bid:", HighestBid[I]
	END IF
NEXT I

OUTPUT "Number of items sold:", CountSold
OUTPUT "Number of items with bids that were unsold:", BidsButUnsold
OUTPUT "Number of items with no bids:", CountNoBids
OUTPUT "Revenue of the auction company:", AuctionRevenue

A check for uniqueness

Use this code sample to check if all of the elements in a certain array are unique. Assume that the array’s name is Array. Replace this with the name of whatever you want to check, like ItemNumbers or BuyerNums, etc… I’m also assuming that there are N items in Array.

Array[1:N]

Unique = True
FOR I = 1 TO N
	PRINT "Input the value of array element", I
	INPUT Array[I]
	FOR J = 1 TO (I - 1)
		IF (Array[I] = Array[J]) THEN
			Unique = False
			J = I + 1
		END IF
	NEXT
NEXT

IF Unique THEN
	PRINT "All of the elements in the array were unique"
ELSE
	PRINT "There are elements in the array which are repeated"
END IF
