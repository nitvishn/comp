Uses of hexadecimal

  1. Memory dump.
    • When memory contents are output to a printer or a monitor.
    • Easier to read in hexadecimal than in binary.
  2. HyperText Markup Language (HTML).
    • Used when writing & developing webpages.
    • Hexadecimal is used in some attributes.
    • Example: representing colours.
  3. Media Access Control (MAC).
    • A number which uniquely identifies a device on the internet.
    • Made up of 48 bits, 64 bits is sometimes used, of hexadecimal digits.
    • 2 types of MAC adresses.
      1. Universal Administered Address (UAA).
        • Most common type of MAC address.
        • Set by the manufacturer at the production factory.
      2. Locally Administered Address (LAA).
    • Reasons for changing the MAC address.
      • To access certain software.
      • To bypass a MAC address filter on a router or a firewall.
      • To bypass certain types of network restrictions.
  4. Web addressees.
    • Each character on a keyboard has an American Standard Code for Information Interchange (ACSII) code.
    • This code is represented by hexadecimal values.
    • Can be used to represent websites.
    • % sign is used to denote that hexadecimal is being used.
    • Currently, Unicode is used in place of ASCII as it uses 64 bits.
  5. Assembly & machine code.
    • Examples of low-level languages.
    • Used to directly access the computer memory.
